This sample provides shows how to use SIMD in a Crosswalk application; in this case, to draw an image of a Mandelbrot set. The code is based on this Mandelbrot animation project by Intel's Peter Jensen and Ningxin Hu.

Note that Crosswalk has native support for SIMD on x86 architecture since version 5.34.104.0. You will need a version of Crosswalk with that version or later to be able to run this demo; and you will need a device with an Intel x86 chipset (emulated or physical).

This application is part of the Crosswalk samples download.

The steps for setting up an environment for Crosswalk are explained in detail in the Getting started tutorial.

SIMD on Android

Once you've set up the Crosswalk Android packaging dependencies, follow the steps in Run on Android to install and run the SIMD sample on Android.

The quick version is that you can build the SIMD apk with:

> cd <xwalk_app_template directory>
> python make_apk.py --manifest=<path to crosswalk-samples>/simd/manifest.json

<xwalk_app_template directory> refers to the directory where you downloaded and unpacked Crosswalk Android.

Then install the apk file on Android:

> adb install simd*.apk

SIMD on Tizen

Follow the steps in Run on Tizen to install and run the SIMD sample on Tizen.

Or here's the quick version:

sdb push <path to crosswalk-samples>/simd /home/developer/simd
sdb shell "xwalk /home/developer/simd/"
