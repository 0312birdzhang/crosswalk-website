Ah, the power of WebGL. This sample provides a quick example of integrating ThreeJS into a Crosswalk application.

This application is part of the Crosswalk samples download.

The steps for setting up an environment for Crosswalk are explained in detail in the Getting started tutorial.

WebGL on Android

Once you've set up the Crosswalk Android packaging dependencies, follow the steps in Run on Android to install and run the WebGL sample on Android.

The quick version is that you can build the WebGL apk with:

> cd <xwalk_app_template directory>
> python make_apk.py --package=org.crosswalkproject.example \
    --manifest=<path to crosswalk-samples>/webgl/manifest.json

<xwalk_app_template directory> refers to the directory where you downloaded and unpacked Crosswalk Android.

Then install the apk file on Android:

> adb install WebGL*.apk

WebGL on Tizen

Follow the steps in Run on Tizen to install and run the WebGL sample on Tizen.

Or here's the quick version:

sdb push <path to crosswalk-samples>/webgl /home/developer/webgl
sdb shell "xwalk --usegl=osmesa /home/developer/webgl/"

Note that you pass the --usegl=osmesa option to enable WebGL on Tizen Crosswalk.
